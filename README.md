# AI Recruitment Platform

Frontend implementation for AI-powered recruitment SaaS platform.

## Tech Stack
- React 18.2
- Vite 5.0
- Tailwind CSS 3.3
- React Router 6.20
- Axios 1.6

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Visit http://localhost:3000

## Build

```bash
npm run build
```

## Pages Implemented

1. Landing Page - Hero and features
2. Login - Authentication
3. Signup - Registration with role selection
4. Candidate Dashboard - Stats and applications
5. CV Upload - Drag-and-drop file upload with job selection
6. Match Result - AI analysis results with skills comparison

## Features

- Responsive design
- Clean SaaS UI
- Role-based routing
- Loading states
- Professional spacing and typography
